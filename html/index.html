HTML - Hipertext Markup Language ou linguagem de marcação de hipertexto

A extensão ".html" ou ".htm" é o que define um arquivo web que pode ser lido pelo navegador.

Como boa prática o arquivo de entrada de um site é o "index.html".

Comentário

A tag <!-- isto é um comentário --> cria uma linha de comentário, muito utilizado por desenvolvedores para não mostrar ou ignorar um trecho de código como também para apontar informações relevantes para outros desenvolvedores quando forem fazer manutenção no código fonte.

Linha horizontal

A tag <hr> cria uma linha horizontal como separador na tela.

Quebra de linha

A tag <br> e usada para forçar uma quebra de linha na tela.

Parágrafo

A tag <p>...</p> é usada para criar parágrafos.

Formatação de texto

Negrito

A tag <strong>...</strong> é usada semânticamente para dar destaque a um trecho de texto importante que se quer destacar.

Como boa prática deve-se evitar usar as tags <b>...</b> ou <big>...</big> para se aplicar essa formatação em negrito

Itálico

A tag <em>...</em> é usada semânticamente para dar destaque a um trecho de texto normalmente usado para palavras estrangeiras, termos técnicos e citações.

Como boa prática deve-se evitar usar as tags <i>...</i> para se aplicar essa formatação em itálico.

Sublinhado

A tag <ins>...</ins> é usada semânticamente para sublinhar um trecho de texto normalmente indica texto adicionado ao original.

Como boa prática deve-se evitar usar as tags <u>...</u> para se aplicar essa formatação como sublinhado.

Tachado

A tag <del>...</del> é usada semânticamente para "riscar" um trecho de texto normalmente indica texto removido do original.

Como boa prática deve-se evitar usar as tags <s>...</s> ou <strike>...</strike> para se aplicar essa formatação em tachado.

Marcação

A tag <mark>...</mark> é usada semânticamente para dar o efeito de um marcador a um trecho de texto.

Small

A tag <small>...</small> é usada semânticamente para dar o efeito de letras pequenas similar aos textos de contratos.

Superescrito

A tag <sup>...</sup> é usada semânticamente para dar o efeito de superescrito como usamos em fómulas matemáticas, x2.

Subscrito

A tag <sub>...</sub> é usada semânticamente para dar o efeito de subscrito como usamos em fómulas químicas, H2O.

Code

A tag <code>...</code> é usada semânticamente para dar o efeito de código fonte.

            //caixa de alerta em JavaScript
            window.alert('Hello World!');

Preformatação

A tag <pre>...</pre> é usada para exibir um trecho de texto com a mesma formatação de qual foi digitado.

        Batatinha quando nasce,
        espalha ramas pelo chão.
        Bebezinho quando dorme,
        põe a mão no coração.

Citação interna

A tag <q>...</q> é usada semânticamente para identificar uma citação interna no prágrafo, colocando-os entre aspas duplas.

Outro dia parei para pensar a respeito de um ditado africano, se quiser ir rápido vá sozinho; mas se quiser ir longe vá acompanhado. Comecei a pensar como posso aplicar isso às minhas experiências.

Citação longa

A tag <blockquote>...</blockquote> é usada semânticamente para identificar uma citação longa, com texto recuado e em fonte menor.

Tenho a audácia de acreditar que os povos em todos os lugares podem fazer três refeições por dia para seus corpos, ter educação e cultura para suas mentes e dignidade, igualdade e liberdade para seus espíritos

Abreviação

A tag <abbr>...<abbr> é usada semânticamente para mostrar com um texto em balão o sentido e abreviações, como HTML.

Sentido de escrita

A tag <bdo>...</bdo> é usada para identificar definir o sentido de escrita de um texto, pode ser invertido.

Link

A tag <a>...</a> é usada para criar hiperligação entre partes de uma página ou entre páginas web diferentes.
